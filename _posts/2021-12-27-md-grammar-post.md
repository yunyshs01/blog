# 마크다운 기본 문법

## 1. 제목

제목을 표시할 때에 사용한다.  
\# 을 여러 개 붙여서 사용할 수 있다.  
\# 뒤에 공백을 사용하면 띄어 쓸 수 있다.  
6개까지 사용할 수 있다.

```md
# 마크다운 기본 문법

## 1. 제목
제목을 표시할 때에 사용한다.  
\# 을 여러 개 붙여서 사용할 수 있다.  
\# 뒤에 공백을 사용하면 띄어 쓸 수 있다.  
6개까지 사용할 수 있다.
```

## 2. 문단

마크다운은 기본적으로 한개의 개행문자는 인식하지 않는다.
하나의 개행문자만 사용하면 하나의 문장으로 인식한다.

두개의 개행문자를 사용하면 문단이 분리된다.  
한 문단안에서 개행을 하고싶다면, 공백문자를 두개 입력 후 개행한다.

```md

마크다운은 기본적으로 한개의 개행문자는 인식하지 않는다.
하나의 개행문자만 사용하면 하나의 문장으로 인식한다.

두개의 개행문자를 사용하면 문단이 분리된다.vv
한 문단안에서 개행을 하고싶다면, 공백문자를 두개 입력 후 개행한다.


```

## 3. 구분선

내용을 구분하고 싶을 때에 사용한다.

----

****

표준에 따라 여러가지로 사용 가능하다.
